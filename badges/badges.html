Badges

In Bootstrap ,a badge is a small count or a label used to highlight information like notifications, count or status

Applications of Badges

  • For notification
  • Satus indicators
  • Labels or Tages
  • Pill Badges

Basic Badges

The basic badge can be used to show a notification like new messages New

Badges in Buttons
